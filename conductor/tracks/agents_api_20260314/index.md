# Track Index: External Agents API (Agent Factory)

This track exposes Oricli-Alpha's `AgentProfile` system via the Native REST API. This allows external applications (HUDs) to dynamically craft, update, and deploy specialized agents that combine specific mindsets (Skills), guardrails (Rules), and tool access (Modules).
