# Thynaptic RFC Index

This document tracks all RFCs (Request for Comments) for core architecture changes in Mavaia Core.

## RFC Status Legend

- **DRAFT** - RFC is being written, not yet complete
- **REVIEW** - RFC is complete and ready for validation
- **APPROVED** - RFC has passed automated validation and is approved
- **IMPLEMENTED** - Changes from RFC have been implemented
- **REJECTED** - RFC was rejected (with reason documented)

## How to Add a New RFC

1. Create RFC file using `RFC_TEMPLATE.md`
2. Assign next sequential RFC number (check existing RFCs)
3. Complete all required sections
4. Add entry to this index
5. Set status to APPROVED when validation passes

## RFC Numbering

RFCs are numbered sequentially: `RFC-YYYY-NNN`
- YYYY: Year (e.g., 2025)
- NNN: Sequential number (001, 002, etc.)

The next available RFC number should be determined by checking existing RFCs in this directory.
